# Web Audit Checklist (CRO + Competitive)

> Default checklist used by `/web-audit`. Curated public CRO heuristics. Edit to use your own checklist; re-run `/onboard-research` Phase 4 to swap.

## Above the fold (first screen)

- [ ] Value proposition is clear in ≤ 7 words
- [ ] Primary CTA visible without scrolling
- [ ] Hero image or video supports the value prop (not decorative)
- [ ] Page loads visually complete in < 2 seconds
- [ ] No autoplay audio or surprise modals

## Information hierarchy

- [ ] F-pattern or Z-pattern reading flow respected
- [ ] One primary action per section (no choice overload)
- [ ] Headings tell the story alone (skim test passes)
- [ ] Body copy ≤ 4 sentences per paragraph (mobile-readable)

## Trust + credibility

- [ ] Social proof visible (logos, counts, named testimonials)
- [ ] Specific numbers, not vague claims ("12,400 customers" not "thousands")
- [ ] Founder or team named with real photo
- [ ] Privacy / security indicators where relevant (SSL, badges, certifications)

## Conversion mechanics

- [ ] CTA button color contrasts with rest of page
- [ ] CTA copy is specific ("Start free trial" not "Submit")
- [ ] Form fields kept to absolute minimum (3 or fewer for top-of-funnel)
- [ ] No required fields beyond what's needed to move forward

## Mobile + accessibility

- [ ] Tap targets ≥ 44×44 px
- [ ] Text legible without zoom (≥ 16px body)
- [ ] No horizontal scroll on common mobile viewports
- [ ] Color contrast meets WCAG AA (4.5:1 for body text)

## Competitive position

- [ ] Pricing is visible (or "request quote" is justified)
- [ ] Differentiator vs top 3 competitors stated explicitly
- [ ] Unique mechanism or methodology named (not generic claims)
- [ ] Recent activity signals (latest blog post, last update date)

## Off-page signals (check if accessible)

- [ ] Domain authority / backlink quality (use SimilarWeb or similar if connector available)
- [ ] Search ranking for key terms (Perplexity for current SERP snapshot)
- [ ] Social proof on third-party platforms (G2, Capterra, Product Hunt)
